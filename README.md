# useful-mini-library
Полезные маленькие утилиты
Полезные маленькие утилиты В процессе программирования часто возникает необходимость в маленьких функциях, выполняющих полезные задачи, например, разбиение чисел на группы разрядов с использованием заданных программистом разделителей, или вывод чисел в двоичной системе счисления, также с использовванием заданных разделителей (например, апострова). В прикладных научных или образовательных программах бывает нужно постоить гисограммы распределений, расчитать свертки, и пр.
