# useful-mini-library
# Полезные маленькие утилиты

## В процессе программирования часто возникает необходимость в маленьких функциях, выполняющих полезные задачи, например, разбиение чисел на группы разрядов с использованием заданных программистом разделителей, или вывод чисел в двоичной системе счисления, также с использовванием заданных разделителей (например, апострофа).  

## В прикладных научных или образовательных программах бывает нужно постоить гистограммы распределений, расчитать свертки, и пр.  

## При профилировании программ часто необходим многофункциональный таймер. Их много, здесь педлагается накая адаптация, сделанная с повторным использованием существующего кода.

## В приложениях компьютерного зрения часто необходимо использовать преобразования Фурье. 

Маленький проект на Python показывает, как это можно в принципе сделать
